For developers,
In order to get email password reset working on localhost, declare environmental variables EMAIL_USER and EMAIL_PASS on local computer
To run code in the RookieDraft directory, install the following (listed by requirements.txt):
-Django
-Pandas
-espn-api
-django-crispy-forms
-Pillow

In order to import league to the website, user's league must be made public on ESPN for the 2020 season and the user must register for an account

Website is currently deployed at dynastydraft.herokuapp.com
